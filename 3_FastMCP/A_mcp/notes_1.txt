=======What is MCP?==========
- Is a way of connecting your client(co-pilot, cursor, claude desktop, Agent) 
to tools e.g Db, Email, or internal documentation.
- MCP provides a common language to connect and build tools
to be used by agents.
Go to:
	cursor.directory/mcp
Then:
	 mcp.run(transport="streamable-http")
The transport is a method that is used to send data between the
client and the server in our case http.
When testing locally use:
	mcp.run(transport = "stdio")
